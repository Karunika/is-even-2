# is-even-2
## Installation

__NPM__ <br/>
`npm i is-even-2`

## Usage

```js
// import
const { isEven } = require("is-even-2");

isEven(0); // true
isEven(5); // false
isEven(183629); // "Tired. Figure it out on your own."
```

## Testing
Simply run `jest` to run unit tests.
